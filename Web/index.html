Sobre el proyecto

TaskMaster es una aplicación de gestión de tareas personales que permite organizar tareas por categorías y estados. Además, se pretende que las tareas puedan ser compartidas entre diversos usuarios y que cada uno de estos pueda tener diversos permisos asignados en cada tarea.
